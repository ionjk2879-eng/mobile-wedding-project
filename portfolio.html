실시간 WYSIWYG 에디터

Data Flow

1

Editor Input

에디터에서 입력 → useState(InvitationData) 상태 업데이트 → 프리뷰 실시간 렌더링

2

Save to Firebase

저장 버튼 클릭 → Firestore에 slug 기반 문서로 저장

3

Guest View

하객이 /w/slug 접속 → Firestore에서 데이터 로드 → 동일한 Preview 컴포넌트로 렌더링

4

RSVP Collection

하객 RSVP 제출 → Firestore 서브컬렉션에 응답 저장

5

Admin Analytics

관리자 페이지에서 참석 통계 조회 및 응답 목록 관리

Engineering Highlights

프로젝트에서 활용한 주요 기술적 접근

Theme System

CSS-in-JS 없이 구현한 테마

CSS 변수와 클래스 전환만으로 8개 테마를 적용합니다.

  • --wedding-main, --wedding-bg 등 CSS 변수 활용
  • SVG 필터 기반 배경 재질 효과
  • @keyframes 파티클 애니메이션 (라이브러리 미사용)
Scroll Interaction

IntersectionObserver 기반 인터랙션

에디터 섹션 자동 하이라이트와 스크롤 등장 효과를 구현합니다.

  • ScrollRootContext로 컨테이너 참조 공유
  • ScrollReveal HOC 분리 설계
  • 에디터-프리뷰 동기 스크롤
External API

Kakao Maps + Daum Postcode 연동

주소 검색부터 교통정보 자동 생성까지 연결합니다.

  • 주소 → 좌표 변환 (Geocoder)
  • 주변 지하철/버스 정류장 자동 탐색
  • 도보 거리 계산 및 교통정보 자동 생성
Layout

반응형 에디터 레이아웃

데스크탑 분할 뷰와 모바일 시뮬레이션을 제공합니다.

  • 에디터(700px) + 프리뷰(450px) 분할
  • 모바일 프리뷰 최대 480px 폭 제한
  • 전체화면 프리뷰 모드 전환
