# 💍 영구 보관용 모바일 청첩장 빌더 (Mobile Wedding Invitation Builder)

> **"소중한 순간을 영원히, 나만의 손길로 만드는 모바일 청첩장"**  
> 고가의 모바일 청첩장 서비스들이 제공하는 기간 제한(2주~3개월) 문제를 해결하기 위해, 사용자가 직접 커스터마이징하고 평생 무료로 영구 보관할 수 있는 웹 기반 빌더 툴입니다.

---

## 🌟 프로젝트 개요 (Project Overview)

- **목적**: 기간 제한 없는 영구 소장용 모바일 청첩장 제작 및 실시간 편집 툴 구현
- **주요 타겟**: 합리적인 비용으로 개성 있는 청첩장을 제작하고 싶은 예비 부부
- **핵심 가치**: 데이터 영구 보관, 직관적인 UI/UX, 고품격 웨딩 에스테틱 제공

---

## 🛠 기술 스택 (Tech Stack)

- **Frontend**: `React 18`, `TypeScript`, `Vite`
- **Styling**: `Vanilla CSS (Inline & Global)`, `Framer Motion` (애니메이션)
- **Icons**: `Lucide React`
- **API**: `Daum Postcode API` (주소 검색 및 자동 입력)
- **Deployment**: `GitHub Pages` (권장), `Vercel`, `Netlify`

---

## ✨ 핵심 기능 (Key Features)

### 1. 실시간 라이브 프리뷰 (Live Preview)
- **WYSIWYG 환경**: 왼쪽 에디터에서 정보를 입력하면 오른쪽 스마트폰 프레임에 즉시 반영됩니다.
- **Narrative Flow**: 실제 청첩장의 읽기 순서(인사말 -> 개인 메시지 -> 장소 -> 정보)와 에디터의 입력 순서를 일치시켜 사용자 인지 부하를 최소화했습니다.

### 2. 고품격 웨딩 에스테틱 (Wedding Aesthetic)
- **디자인 프리셋**: 샴페인 아이보리, 뮤트 샌드, 로즈 베이지 등 최신 트렌드를 반영한 컬러 팔레트 적용.
- **폰트 커스터마이징**: 클래식한 명조체부터 감성적인 필기체까지 5종의 구글 폰트 선택 옵션 제공.
- **반응형 스케일링**: 브라우저 해상도에 따라 미리보기 화면이 자동으로 조절되어 다양한 작업 환경 지원.

### 3. 스마트 편의 기능
- **주소 검색 API**: 예식장 주소 검색 시 주소와 건물명이 자동 입력되어 번거로움을 덜어줍니다.
- **계좌번호 복사**: 하객들이 번거롭게 외울 필요 없이 클릭 한 번으로 송금 계좌를 복사할 수 있습니다.
- **배경음악(BGM)**: MP3 URL 입력을 통해 청첩장에 감성적인 배경음악을 삽입할 수 있습니다.
- **연락처 연동**: 전화/문자 아이콘 클릭 시 기기 기본 앱과 즉시 연동됩니다.

## 📐 시스템 아키텍처 (Architecture)

```text
src/
├── components/
│   ├── Editor/
│   │   └── EditorContainer.tsx (전체 입력 폼 및 상태 변경 제어)
│   └── Preview/
│       ├── Hero.tsx            (메인 비주얼)
│       ├── Greeting.tsx        (공식 초대 문구)
│       ├── PersonalMessage.tsx (신랑/신부 개별 메시지)
│       ├── Gallery.tsx         (웨딩 사진 갤러리)
│       ├── Location.tsx        (지도 및 길찾기 안내)
│       ├── Money.tsx           (축의금 및 계좌 정보)
│       └── Share.tsx           (BGM 플레이어 및 공유 기능)
├── types.ts                    (InvitationData 인터페이스 정의)
├── App.tsx                     (중앙 상태 관리 및 레이아웃 구성)
└── index.css                   (글로벌 테마 및 Reset CSS)
```

## 💡 개발 포인트 및 문제 해결 (Engineering Note)

- **데이터 기반 아키텍처**: 모든 UI가 `InvitationData`라는 단일 상태(State)에 의존하도록 설계하여 코드 유지보수성을 높였습니다.
- **사용자 경험 최적화**: 에디터와 프리뷰 패널에 독립적인 스크롤 시스템을 적용하여, 긴 내용을 입력할 때도 작업 흐름이 끊기지 않도록 개선했습니다.
- **영구 보관 최적화**: 외부 라이브러리 의존도를 낮추고 표준 웹 기술(TypeScript + CSS) 위주로 제작하여, 시간이 지나도 깨지지 않는 안정적인 렌더링을 보장합니다.

## 🚀 실행 및 배포 (Setup & Deployment)

1. **로컬 실행**
   ```bash
   npm install
   npm run dev
   ```
2. **배포용 빌드**
   ```bash
   npm run build
   ```
3. **배포**: 생성된 `dist` 폴더의 정적 파일들을 GitHub Pages 등에 업로드하여 평생 무료로 호스팅할 수 있습니다.
